emissions {Simple}R Documentation

CO2 emissions data and Gross Domestic product for 26 countries

Description

The emissions data frame has 26 rows and 3 columns.

Data set for 1999

lists GDP, GDP/capita and CO2 emissions.

Usage

data(emissions)

Format

This data frame contains the following columns:

GDP
a numeric vector
perCapita
a numeric vector
CO2
a numeric vector

Source

www.grida.no for CO2, www.mrdowling.com for GDP.

Prompted by a plot appearing in a June 2001 issue of the NY Times.

Examples

data(emissions)
plot(emissions)
