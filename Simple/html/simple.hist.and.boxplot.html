Description

Simple function to plot both histogram and boxplot to compare

Usage

Arguments

x vector of univariate data
... Arguments passed to the hist function

Value

Just prints the two graphs

Author(s)

John Verzani

Examples

x<-rnorm(100)
simple.hist.and.boxplot(x)
